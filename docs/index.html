Everything you need to process images

Four powerful tools in one app, all with batch processing and live progress.

Convert

Batch convert to JPEG, PNG, WebP, AVIF, GIF, BMP, TIFF, ICO, and PDF. Set quality and resize percentage per run.

  • 9 output formats
  • Quality control for lossy formats
  • Resize by percentage

Crop

Crop images to exact aspect ratios with a visual gravity picker. Handles portrait and landscape orientations automatically.

  • 8 preset ratios (1:1, 16:9, 4:3 …)
  • 9-position crop gravity
  • Visual ratio preview

Watermark

Overlay text or image watermarks anywhere on a photo. Control font, size, colour, opacity, and position independently.

  • Text or image overlay
  • 9-position gravity grid
  • Opacity & offset control

Background Removal

Remove backgrounds by colour (flood-fill) or with AI models. The AI engine is fully bundled — no Python install needed.

  • Colour / flood-fill method
  • AI removal (rembg, bundled)
  • 9 model choices

Get started in seconds

1

Download the installer

Grab MagickGUI Setup 1.0.0.exe from the releases page. ImageMagick 7 and rembg are bundled — no separate installs needed.

Download v1.0.0
2

Run the installer

Double-click the .exe and follow the NSIS installer. Choose your install directory. Creates a desktop shortcut and Start Menu entry.

3

Drop images and go

Open MagickGUI, pick a tool, drag your images in, set your options, and click Start. AI models download automatically on first use.

System Requirements

OS Windows 10 / 11 (x64)
Disk ~400 MB for app + space for AI models (45–350 MB each)
RAM 4 GB minimum, 8 GB recommended for AI models
Internet Required only for first-use AI model download

Build from source

Requires Node.js 18+, npm 9+, and Python 3.10–3.12 on Windows.

# Clone and install dependencies git clone https://github.com/saufik-ramadhan/magick-gui.git cd magick-gui npm install
# Download ImageMagick + build rembg.exe (run once) npm run prepare-resources
# Build the NSIS installer npm run package
# Or run in dev mode (needs magick + rembg on PATH) npm run dev
